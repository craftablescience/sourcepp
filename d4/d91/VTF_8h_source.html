85  * === EASY DIFFICULTY WRITER API ===
86  *
87  * Use a static helper function to create a VTF in one function call - VTF::create
88  *
89  * === MEDIUM DIFFICULTY WRITER API ===
90  *
91  * Constructing a VTF instance from existing VTF data will let you modify that data.
92  *
93  * This class has methods that should be called in a particular order
94  * when creating a VTF from scratch, or your output VTF may look incorrect:
95  *
96  * 0. Construct an empty VTF instance
97  * 1. Set the version (7.4 is the default) - VTF::setVersion
98  * 2. Set FLAG_SRGB (optional, read for image resizing) - VTF::setFlags
99  * 3. Set the image resize methods (optional) - VTF::setImageResizeMethods
100  * 4. Set the base image (mip 0, frame 0, face 0, slice 0) - VTF::setImage
101  * 5. Compute mips (optional) - VTF::computeMips
102  * 6. Set the output format (optional) - VTF::setFormat
103  *
104  * After these methods are called, the other writer methods in the class should work as expected.
105  */

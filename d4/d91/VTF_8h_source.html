92/*
93 * === EASY DIFFICULTY WRITER API ===
94 *
95 * Use a static helper function to create a VTF in one function call - VTF::create
96 *
97 * === MEDIUM DIFFICULTY WRITER API ===
98 *
99 * Constructing a VTF instance from existing VTF data will let you modify that data.
100 *
101 * This class has methods that should be called in a particular order
102 * when creating a VTF from scratch, or your output VTF may look incorrect:
103 *
104 * 0. Construct an empty VTF instance
105 * 1. Set the version (7.4 is the default) - VTF::setVersion
106 * 2. Set FLAG_SRGB (optional, read for image resizing) - VTF::setFlags
107 * 3. Set the image resize methods (optional) - VTF::setImageResizeMethods
108 * 4. Set the base image (mip 0, frame 0, face 0, slice 0) - VTF::setImage
109 * 5. Compute mips (optional) - VTF::computeMips
110 * 6. Set the output format (optional) - VTF::setFormat
111 *
112 * After these methods are called, the other writer methods in the class should work as expected.
113 */
